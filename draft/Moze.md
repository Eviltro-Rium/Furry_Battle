# 🐼🟢 Moze

- **初始血量**：100
- **被动**：每次受到伤害时，可选择消耗若干层【守护】来减免最终受到的伤害，每消耗1层【守护】减免1点🗡️。【守护】不能减免【流血】的伤害。

## ⚔️ 进攻阶段

| 牌 | 效果 |
|---|------|
| 1️⃣ | 3点🗡️ |
| 2️⃣ | 2点🗡️，获得1层【守护】；若打出的牌是🟢则再获得1层【守护】 |
| 3️⃣ | 1点🗡️，恢复1❤️，获得1层【守护】 |
| 4️⃣ | 打出1张数字牌（对应技能不生效），获得对应数字点【守护】，然后弃掉该牌（跳过防御） |
| 5️⃣ | 从攻击目标手牌中抽取1🃏（入判定区），若为🟢/⚪/⚫→4点🗡️；否则→恢复2❤️，获得1层【守护】（跳过防御）。无论判定如何都要将该牌加入自己手牌 |
| 6️⃣ | 造成2+【守护】层数点🗡️；如果没有【守护】则变为不可防御 |
| 7️⃣ | 3点🗡️，清除所有⬇️debuff，每清除1层debuff+1点🗡️ |
| 0️⃣ | 获得3层【守护】，5点🗡️，抽1🃏 |

## 🛡️ 防御阶段

| 牌 | 效果 |
|---|------|
| 1️⃣ | 防御½（向上取整）点🗡️，获得1层【守护】 |
| 2️⃣ | 反击1+½×【守护】层数（向上取整）点🗡️ |
| 3️⃣ | 获得1层【守护】，恢复1/2*【守护】层数（向上取整）点❤️ |
| 0️⃣ | 防御½（向上取整）点🗡️，获得2层【守护】，反击2×【守护】层数点🗡️ |

## 【守护】机制

- 【守护】是正面buff，上限5层
- 每次受到伤害结算时，弹窗选择消耗0~N层【守护】（N=min(守护层数, 剩余伤害)），每层减免1点🗡️
- 【守护】不能减免【流血】的伤害
- ✨✨超级净化会清除所有buff，包括【守护】
- 其他清除debuff的技能（Ryan 6️⃣/0️⃣、Blaze 6️⃣、Moze 7️⃣）只清除⬇️debuff，不清除【守护】

## 举例

对方造成9点🗡️，Moze有3层【守护】，Moze打出0️⃣防御½（5点伤害），获得2层【守护】（共5层），反击2×5=10点🗡️。防御结算后，Moze选择消耗4层【守护】来免去4点🗡️，Moze最终受到0点🗡️。如果Moze有【流血】仍需额外受到【流血】层数点🗡️。
