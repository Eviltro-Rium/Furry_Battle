# 🐺🟡 Saiki

- **初始血量**：80
- **被动**：⚔️阶段打出🟡牌（含指定为黄色的白牌/黑牌），对对手施加1层🩸（流血最多3层，有该debuff的角色触发防御技能时受到1点额外独立伤害）

## ⚔️ 进攻阶段

| 牌 | 效果 |
|---|------|
| 1️⃣ | 4点🗡️ |
| 2️⃣ | 3点🗡️，恢复1点❤️ |
| 3️⃣ | 2点🗡️，抽对手手牌1🃏（可选择弃掉或加入手牌） |
| 4️⃣ | 5点🗡️，若对手有🩸则不可防御 |
| 5️⃣ | 若❤️≤20→恢复4❤️；若20<❤️≤50→4🗡️跳过防御；若❤️>50→4🗡️+抽对手1🃏 |
| 6️⃣ | 再打1张数字🃏入判定区，对应点数×1.5（向上取整）🗡️，若为🟡牌施加1层🩸，判定牌放入弃牌库底 |
| 7️⃣ | 2+2×对手🩸层数🗡️，恢复等量❤️ |
| 0️⃣ | 施加1层🩸，3×对手🩸层数+1点🗡️，恢复双方🩸层数之和❤️ |

## 🛡️ 防御阶段

| 牌 | 效果 |
|---|------|
| 1️⃣ | 防御至多3点🗡️ |
| 2️⃣ | 反击3🗡️，⚔️结束后施加1层🩸 |
| 3️⃣ | 翻开牌库顶1张判定：🟡/⚫/⚪→防御所有🗡️（不免疫⬇️debuff）；否则无效，判定牌加入手牌 |
| 0️⃣ | 防御所有🗡️，免疫⬇️debuff，双方均摊½（向上取整）🗡️，debuff反弹给进攻者 |

## 备注

- 🗡️=伤害点数，❤️=恢复生命值，🃏=卡牌，🩸=流血debuff
- 流血最多2层，防御时每层造成1点额外独立伤害
- 被动触发条件：打出🟡牌，包括白牌/黑牌指定为黄色的情况
- 4️⃣不可防御条件：对手有流血层数
- 5️⃣根据自身当前❤️分段效果不同
- 6️⃣判定牌技能不生效，只算点数；🟡判定牌额外施加流血
- 7️⃣伤害和恢复均基于对手流血层数
- 0️⃣恢复量为双方流血层数之和
