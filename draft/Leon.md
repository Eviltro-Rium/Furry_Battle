# 🐻‍❄️🔴 Leon

- **初始血量**：100
- **被动**：免疫灼烧。灼烧buff最高4层。拥有该buff的玩家，在进攻结束后，有几层灼烧受到几点伤害，然后去除一层灼烧。

| 4️⃣ | 5🗡️，目标有【灼伤】→跳过防御 |

## 🛡️ 防御阶段

| 牌 | 效果 |
|---|------|
| 1️⃣ | 施加1层【灼伤】给攻击方，恢复2❤️ |
| 2️⃣ | 反击等量🗡️，抽1🃏 |
| 3️⃣ | 格挡½（向上取整）所受🗡️，抽1🃏 |
| 0️⃣ | 攻击方弃掉所有牌，免疫伤害，反击等量🗡️ |


## 备注

- 🗡️=伤害点数，❤️=恢复生命值，🃏=卡牌
- 🧪=药水牌（白色道具牌，恢复5❤️+搭桥，可匹配任意颜色）
- +3🃏=抽3张的白色道具牌（搭桥，可匹配任意颜色）
- 搭桥可连续使用（黑牌→🧪→+3→黑牌…）
- 数字牌和道具牌互斥
