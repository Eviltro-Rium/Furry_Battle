# 🐶🔴 Blaze

- **初始血量**：85
- **被动**：当你【灼伤】时，你的⚔️阶段的所有攻击+1点🗡️（参与🛡️的计算），该伤害加成不能叠加

## ⚔️ 进攻阶段

| 牌 | 效果 |
|---|------|
| 1️⃣ | 4点🗡️ |
| 2️⃣ | 对攻击目标造成2点🗡️（不可防御），然后对自己施加2层【灼伤】 |
| 3️⃣ | 造成3点🗡️，然后对自己和对手施加1层【灼伤】|
| 4️⃣ | 从攻击目标的手牌中抽取一张牌（入判定区），施加该牌对应点数🗡️（道具牌视为4️⃣点），然后弃掉该牌；若该牌为0️⃣，则不造成伤害（跳过防御），对双方各施加1层【灼烧】然后把这张0️⃣加入自己手牌 |
| 5️⃣ | 对自己施加1层【灼伤】，然后造成2×【灼伤】层数点🗡️（含被动加成） |
| 6️⃣ | 恢复1.5×自身【灼烧】层数（向上取整）点❤️，并对对手施加1层【灼伤】，清除所有【灼烧】（跳过防御） |
| 7️⃣ | 先对双方各施加2层【灼伤】，然后对目标造成1.5×场上所有【灼伤】层数（向上取整）点🗡️（含被动加成） |
| 0️⃣ | 对除自己以外，所有玩家造成6点🗡️伤害（不可防御），并施加2层【灼伤】|

## 🛡️ 防御阶段

| 牌 | 效果 |
|---|------|
| 1️⃣ | 恢复2+自身【灼伤】层数点❤️，然后对双方各施加1层【灼烧】 |
| 2️⃣ | 抽取1🃏（放入判定区），反击该牌点数（道具牌视为4️⃣点，额外施加一层【灼伤】）点🗡️，加入手牌 |
| 3️⃣ | 对进攻玩家施加2层【灼伤】，反击场上所有【灼伤】层数点🗡️（先施加灼伤再结算伤害） |
| 0️⃣ | 对进攻玩家施加4层【灼伤】，防御1/2（向上取整）点🗡️，恢复场上所有【灼烧】数+3点❤️（先施加灼伤再结算回血） |

## 备注

- 被动加成在1️⃣2️⃣3️⃣4️⃣5️⃣6️⃣7️⃣的攻击伤害中生效
- 5️⃣和7️⃣先施加灼伤再计算伤害，因此被动加成保证生效
- 4️⃣抽到0️⃣时施加灼烧而非造成伤害，且跳过防御
- 防御1️⃣2️⃣3️⃣0️⃣均涉及灼伤施加，先施加灼伤再结算后续效果
