# Serenity

- **初始血量**：80
- **被动**：免疫【冷冻】；❤️<30时进入【嗜血】态（非buff，无法清除）；❤️>=30时正常状态，恢复时额外+1❤️

## 进攻阶段

| 牌 | 效果 |
|---|------|
| 1️⃣ | 恢复3点❤️ |
| 2️⃣ | 3点🗡️；【嗜血】时改为5点🗡️ |
| 3️⃣ | 2点🗡️，恢复2点❤️ |
| 4️⃣ | 5点🗡️；【嗜血】时先施加1层【流血】再造成5点🗡️ |
| 5️⃣ | 翻开牌库顶1张牌：黄/绿→恢复4❤️；否则→对对手5点🗡️；弃掉该牌 |
| 6️⃣ | 6点🗡️；【嗜血】时不可防御 |
| 7️⃣ | 对自己2点🗡️+对对手5点🗡️（不可防御）；【嗜血】时自己不受伤害 |
| 0️⃣ | 恢复1❤️，弃掉所有手牌，每弃1张+3❤️（上限9），重抽4张继续⚔️；【嗜血】时对手也弃所有手牌并抽原数-1张 |

## 防御阶段

| 牌 | 效果 |
|---|------|
| 1️⃣ | 防御至多3点🗡️；【嗜血】时额外恢复实际防御点数❤️ |
| 2️⃣ | 对进攻方施加1层【流血】，每层流血吸取2点❤️ |
| 3️⃣ | 防御½（向上取整）点🗡️；【嗜血】时多防御2点 |
| 0️⃣ | 翻开牌库顶1张牌：黄→进攻方⚔️结束后❤️减半（非伤害）；否则→免疫所有伤害和debuff；弃掉该牌 |

## 备注

- 【嗜血】态由血量自动触发/解除，非buff不可净化
- 免疫冷冻：任何来源的冷冻效果均无效
- 正常态恢复+1：所有❤️恢复效果额外+1（包括药水、技能恢复等）
- 5️⃣判定：黄/绿牌恢复，其他牌造成伤害
- 7️⃣嗜血时自伤为0，但仍对对手造成不可防御伤害
- 0️⃣嗜血时对手弃牌后抽原手牌数-1张
- 防御0️⃣黄牌效果：❤️减半不是伤害，不可防御/免疫
