# 🔵 Chan

- **初始血量**：80
- **被动**：在进攻回合开始前可以抽取一张牌。（回合结束时手牌依旧不能超过5张）

## ⚔️ 进攻阶段

| 牌 | 效果 |
|---|------|
| 1️⃣ | 1点🗡️，施加【冷冻】（跳过防御） |
| 2️⃣ | 4点🗡️ |
| 3️⃣ | 2点🗡️，抽取1🃏 |
| 4️⃣ | 抽取对手1张手牌，选择用你的1张手牌强制交换；若不交换，则该玩家弃掉该牌并造成2点🗡️（跳过防御） |
| 5️⃣ | 消耗2❤️，查看【牌库顶】5🃏，并交换顺序后放回【牌库顶】，然后抽取2🃏（跳过防御） |
| 6️⃣ | 5点🗡️，并抽取1🃏，若是🔵/⚪/⚫则不能被防御（跳过防御），无论判定结果如何都要将判定牌加入手牌 |
| 7️⃣ | 6点🗡️，选择对手一张牌放入判定区展示，可以选择放入弃牌库，也可以选择加入自己手牌 |
| 0️⃣ | 7点🗡️，施加【冷冻】，抽取1🃏 |

## 🛡️ 防御阶段

| 牌 | 效果 |
|---|------|
| 1️⃣ | 格挡½（向上取整）点🗡️ |
| 2️⃣ | 反击2点🗡️，并施加【冷冻】 |
| 3️⃣ | 翻开【牌库顶】1🃏，恢复½对应点数❤️（向上取整，道具牌算0点），然后加入手牌 |
| 0️⃣ | 防御所有伤害（不免疫⬇️buff），并反击所受伤害½点🗡️，进攻方立刻结束回合 |

## 【冷冻】效果

- ⬇️buff，有该buff的玩家不能防御🔵属性攻击（包括被指定为蓝色的白色牌）
- 最多叠1层，持续直到被清除

## 备注

- 🗡️=伤害点数，❤️=恢复生命值，🃏=卡牌
- 🧪=药水牌（白色道具牌，恢复5❤️+搭桥，可匹配任意颜色）
- +3🃏=抽3张的白色道具牌（搭桥，可匹配任意颜色）
- 搭桥可连续使用（黑牌→🧪→+3→黑牌…）
- 数字牌和道具牌互斥
